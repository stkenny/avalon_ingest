# AvalonIngest

Gem containing the ingest code from Avalon

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'avalon_ingest'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install avalon_ingest

Then execute:
   
    $ rails generate avalon_ingest:install
